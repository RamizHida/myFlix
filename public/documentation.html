Documentation

myFlix API contains information about movies, actors, and directors. This information is stored in a database. It includes basic CRUD functionality.

Business Logic Endpoint URL HTTP Method Request Response
Get a list off all movies /movies GET None A JSON object holding data on all movies. Example: [{ movie-title: 'The Conguring', genre: 'Horror', director: 'James Wan', image-URL: 'moviephotos.com/conguring', featured: 'true' }, { movie-title: 'Scarface', genre: 'crime drama', director: 'Brian De Palma Wan', image-URL: 'moviephotos.com/scarface', featured: 'true' }]
Return data about a single movie by title /movies/[movie-title] GET None A JSON object holding data on a single movie. Example: { description: 'A determined Cuban immigrant takes over a drug cartel and succumbs in greed.', movie-title: 'Scarface', genre: 'crime drama', director: 'Brian De Palma Wan', image-URL: 'moviephotos.com/scarface', featured: 'true' }
Return data about a genre by name/title /movies/[movie-title]/genre GET None A JSON object holding data on movie genre and description. Example: { description: 'A determined Cuban immigrant takes over a drug cartel and succumbs in greed.', movie-title: 'Scarface', genre: 'crime drama' }
Return data about a director by name /movies/[director] GET None A JSON object holding data on movie director. Example: {bio: 'John Smith was born in 1907, he attended...', birthday: 1907, death-year: 1988 }
Allow new users to register /users POST A JSON object holding data about the user. Example : {username: 'Sam Iam', email: 'samiam@gmail.com', ID: 1234 } A JSON object holding data about the user that was added, including ID. Example: {ID: 12345, username: 'Sam Iam', email: 'samiam@gmail.com', ID: 1234 }
Allow users to update their user info /users/[id] PUT None A text message indicating the update was successful
Allow users to add a movie to their list of favorites /users/[id]/[favorite-movies] POST None A text message indicating the movie has been added
Allow users to remove a movie from their list of favorites /users/[id]/[favorite-movies] DELETE None A text message indicating the movie has been removed
Allow users to deregister /users/[id] DELETE None A text message indicating the account linked with the user has been deleted
